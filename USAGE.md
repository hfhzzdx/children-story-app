# 使用说明

本文档详细说明"小墩子故事屋"应用的使用方法和功能。

## 应用概述

"小墩子故事屋"是一款专为3-6岁儿童设计的智能故事播放应用。通过丰富的故事内容和友好的交互界面，帮助儿童在娱乐中学习成长。

## 主要功能

### 1. 故事播放
- **播放控制**：播放、暂停、快进、快退
- **定时关闭**：设置播放时长，自动停止
- **后台播放**：锁屏或切换到其他应用时继续播放
- **播放历史**：记录播放进度，下次继续

### 2. 故事库管理
- **分类浏览**：按故事类型浏览
- **搜索功能**：按标题或关键词搜索
- **收藏功能**：标记喜欢的故事
- **最近播放**：快速访问最近听过的故事

### 3. 文件导入
- **支持格式**：PDF、TXT、EPUB
- **批量导入**：一次导入多个文件
- **自动分类**：根据内容智能分类
- **内容解析**：提取文本生成音频

### 4. 家长控制
- **时间限制**：设置每日使用时长
- **内容过滤**：过滤不适合的内容
- **播放统计**：查看使用报告
- **设置管理**：调整应用设置

## 快速入门

### 首次使用
1. **启动应用**：点击应用图标启动
2. **权限授权**：根据提示授予必要权限
3. **初始设置**：设置儿童年龄和偏好
4. **开始探索**：浏览故事库开始使用

### 基本操作
1. **播放故事**：
   - 在故事库中选择一个故事
   - 点击播放按钮开始播放
   - 使用控制栏调整播放

2. **导入故事**：
   - 点击"导入"按钮
   - 选择文件或文件夹
   - 等待导入完成

3. **设置定时**：
   - 播放界面点击定时图标
   - 选择关闭时间（15/30/60分钟）
   - 或选择"播完当前故事"

## 故事分类说明

### 1. 通话故事
- **特点**：简单对话形式，适合语言学习
- **时长**：3-5分钟
- **示例**：《小兔子打电话》、《小熊的生日派对》

### 2. 绘本故事
- **特点**：图文结合，视觉体验好
- **时长**：5-8分钟
- **示例**：《好饿的毛毛虫》、《猜猜我有多爱你》

### 3. 睡前故事
- **特点**：节奏舒缓，帮助入睡
- **时长**：8-12分钟
- **示例**：《月亮晚安》、《小星星的梦》

### 4. 成语故事
- **特点**：传统文化，寓教于乐
- **时长**：4-6分钟
- **示例**：《守株待兔》、《画蛇添足》

### 5. 寓言故事
- **特点**：道德教育，启发思考
- **时长**：5-7分钟
- **示例**：《乌鸦喝水》、《龟兔赛跑》

### 6. 科普故事
- **特点**：知识普及，激发兴趣
- **时长**：6-9分钟
- **示例**：《太阳系旅行》、《小水滴的旅程》

## 文件导入指南

### 支持的文件格式
1. **PDF文件**：
   - 支持文本PDF
   - 支持扫描PDF（需要OCR）
   - 最大文件大小：50MB

2. **TXT文件**：
   - 支持UTF-8编码
   - 支持ANSI编码
   - 最大文件大小：10MB

3. **EPUB文件**：
   - 支持标准EPUB格式
   - 支持带图片的EPUB
   - 最大文件大小：100MB

### 导入步骤
1. **选择文件**：
   - 点击"导入"按钮
   - 选择"从文件导入"或"从文件夹导入"
   - 浏览并选择文件

2. **解析内容**：
   - 应用自动解析文件内容
   - 提取文本和图片
   - 识别故事结构

3. **分类设置**：
   - 自动推荐分类
   - 可手动调整分类
   - 设置年龄范围

4. **生成音频**：
   - 使用TTS生成语音
   - 添加背景音乐
   - 保存到本地库

### 导入注意事项
1. **文件编码**：确保TXT文件使用正确编码
2. **文件大小**：过大文件可能导入失败
3. **内容质量**：确保内容适合儿童
4. **版权问题**：仅导入拥有版权的文件

## 故障排除

### 常见问题

#### 1. 无法播放音频
**可能原因**：
- 音量设置为0
- 设备静音模式
- 音频文件损坏

**解决方案**：
1. 检查设备音量
2. 关闭静音模式
3. 重新导入故事

#### 2. 导入文件失败
**可能原因**：
- 文件格式不支持
- 文件损坏
- 存储空间不足

**解决方案**：
1. 检查文件格式
2. 尝试其他文件
3. 清理存储空间

#### 3. 应用卡顿
**可能原因**：
- 设备内存不足
- 同时运行应用过多
- 应用缓存过多

**解决方案**：
1. 关闭其他应用
2. 清理应用缓存
3. 重启设备

#### 4. 无法后台播放
**可能原因**：
- 系统省电模式
- 应用权限限制
- 系统设置限制

**解决方案**：
1. 检查省电设置
2. 授予后台运行权限
3. 调整系统设置
